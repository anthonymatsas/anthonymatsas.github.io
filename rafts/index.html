Thougth Bucket

Thought Bucket is a small social network I am building with PHP and MySQL. Users can think out loud by posting their most current thoughts. Then other users have the ability to upvote or downvote a thought depending on how they feel about it. Also, users can comment on any thoughts!

Lost is a text based game built with JavaScript. The user is lost on an island and has to find his/her way off of the island by typing commands like: "go north", "go south", "take backpack", "drop weapon" and so on. The main technologies used for this application are Javascript, HTML5 and CSS3.
