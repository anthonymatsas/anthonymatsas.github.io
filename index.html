Project: chicagobulls-watchface
Platform: Pebble Watchface
Link
This is a watchface app for your pebble watch. Currently works on the new pebble time as well, but a colored version will be coming soon. Current features include: location based weather, time, date, battery status and ofcourse the Chicago Bulls logo. This project has helped me learn a lot about pebble development and using C. Also, learned how to efficiently use APIs, while finding the pebble SDK a pleasure to work with. I will definitely be working on further updates on this and I want to create a dedicated pebble time app in the future as well.
