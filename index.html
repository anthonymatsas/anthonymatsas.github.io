Chicago Bulls Watchface

Platform: Pebble
Technology: Javascript, C
Link

This is a watchface app for the OG pebble watch. Currenty works on the pebble time as well, but a colored version is not yet available. Current features include: location based weather, time, date, batter status, and a sweet looking Chicago Bulls logo. This projects has helped me learn a lot about the pebble SDK and gave me a chance to use C. Also, using Javascript to connect to an API was really interesting.
